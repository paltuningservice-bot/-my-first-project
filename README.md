# 🌟 My First Project - مشاريع تقنية وإنسانية

## نبذة
هذا المشروع يحتوي على برامج تقنية وأدوات للمشاريع الإنسانية.

## اللغات المستخدمة
- Python 3.x
- JavaScript (Electron)

## الأدوات المطلوبة
```bash
pip install -r requirements.txt
```

## هيكل المشروع
```
my-first-project/
├── src/              # الكود المصدري
├── docs/             # التوثيق
├── tests/            # الاختبارات
└── README.md
```

## كيف تبدأ
1. ثبّت Python 3.11+
2. شغّل: `pip install -r requirements.txt`
3. اذهب إلى src/ وشغّل البرامج

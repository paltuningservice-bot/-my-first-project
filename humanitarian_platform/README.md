# 🌟 منصة إدارة المشاريع الإنسانية

## نبذة عن المنصة
منصة متكاملة لإدارة المشاريع الإنسانية والتنموية، تساعد المؤسسات في:
- إدارة المشاريع والمستفيدين
- تتبع الأنشطة والمؤشرات
- إنشاء التقارير المالية والتقييمية
- دعم اللغة العربية والإنجليزية

## المتطلبات
- Python 3.8+
- Windows 10/11

## التثبيت

### 1. تثبيت Python
```
1. حمل Python من: https://www.python.org/downloads/
2. اختر Python 3.11 أو أحدث
3. أثناء التثبيت: ✅ Add Python to PATH
```

### 2. تثبيت المكتبات
```bash
cd humanitarian_platform
pip install -r requirements.txt
```

### 3. تشغيل المنصة
```bash
python start.py
```

## هيكل المشروع
```
humanitarian_platform/
├── src/
│   ├── models.py      # نماذج البيانات
│   └── main_app.py    # الواجهة الرئيسية
├── data/              # ملفات البيانات
├── backups/           # النسخ الاحتياطية
├── reports/           # التقارير
└── start.py           # مشغل البرنامج
```

## الميزات الرئيسية

### 📊 لوحة التحكم
- ملخصات الأداء والإحصائيات
- أحدث المشاريع والمستفيدين
- مؤشرات الإنجاز

### 📁 إدارة المشاريع
- إضافة/تعديل/حذف المشاريع
- تتبع الميزانية والجدول الزمني
- متابعة نسبة الإنجاز

### 👥 إدارة المستفيدين
- سجل المستفيدين
- البيانات الديموغرافية
- تتبع الفئات المستهدفة

### 📅 إدارة الأنشطة
- جدولة الأنشطة والفعاليات
- تتبع المهام
- قياس النتائج

### 📊 التقارير
- تقارير تقدم المشاريع
- تقارير مالية
- تقارير تقييم الأداء
- تصدير PDF/Excel

### 💾 النسخ الاحتياطي
- نسخ تلقائي للبيانات
- استعادة البيانات
